Project Github

This project creates animated PacMen. Each time the "Add PacMan" button is pressed, a new PacMan is created. Once satisfied with the amount, the "Start Game" button can be clicked. This will cause the PacMen to start moving in all directions at different velocities. The PacMen also have collision detection when they hit the edge of the screen.

This eyes program allows the user to move the eyes with the movement of the mouse. When you move your mouse around the screen, the eyes follow you!

Project Github

The Real Time Bus Tracker shows the animation of the real route of the Boston MBTA Red Line moving from MIT to Boston Common. The program utilizes JavaScript and Mapbox API, along with HTML and CSS, to render and manipulate a map on a webpage.
